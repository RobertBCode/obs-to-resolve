- Single-file conversion

- Minimalist GUI  
- Fully free

- Presets for popular formats:

## Usage

1. Download the ZIP for your version.  
2. Extract the files.  
3. Run the executable or script (Linux `.AppImage` or `.py` for Python).  
4. Select your input files, choose a preset, and convert!
